# Android Native Libraries

Place precompiled libvips .so files in the appropriate architecture directories:

```
jniLibs/
├── arm64-v8a/
│   └── libvips.so
├── armeabi-v7a/
│   └── libvips.so
└── x86_64/
    └── libvips.so
```

## Download

Download precompiled libraries from:
- https://github.com/AiHrt/libvips-android-build/releases

Or build from source following the libvips documentation.
