# libvips_ffi_windows

Pre-compiled libvips for Windows. Part of the libvips_ffi package family.

## Features

- Pre-compiled libvips binaries for Windows
- Supports x64 architecture

## Usage

```dart
import 'package:libvips_ffi_windows/libvips_ffi_windows.dart';

void main() {
  initVipsWindows();
  
  // Use libvips...
  
  shutdownVips();
}
```

## Note

This package contains placeholder structure. Pre-compiled binaries need to be added before publishing.
