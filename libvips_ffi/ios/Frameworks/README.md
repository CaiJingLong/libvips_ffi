# iOS Frameworks

Place the precompiled `libvips.xcframework` in this directory.

## Download

Download precompiled xcframework from:

<https://github.com/CaiJingLong/libvips_precompile_mobile/releases>

Extract and place `libvips.xcframework` here:

```text
Frameworks/
└── libvips.xcframework/
    ├── Info.plist
    ├── ios-arm64/
    │   ├── Headers/
    │   └── liblibvips.a
    └── ios-arm64-simulator/
        ├── Headers/
        └── liblibvips.a
```
