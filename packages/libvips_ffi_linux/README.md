# libvips_ffi_linux

Pre-compiled libvips for Linux. Part of the libvips_ffi package family.

## Features

- Pre-compiled libvips binaries for Linux
- Supports x64 architecture

## Usage

```dart
import 'package:libvips_ffi_linux/libvips_ffi_linux.dart';

void main() {
  initVipsLinux();
  
  // Use libvips...
  
  shutdownVips();
}
```

## Note

This package contains placeholder structure. Pre-compiled binaries need to be added before publishing.
